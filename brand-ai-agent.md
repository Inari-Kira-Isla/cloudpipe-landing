## Brand AI Agent：四大核心模組

CloudPipe 為每個品牌打造專屬的 AI Agent — 一個真正理解你品牌的「數位靈魂」。

### 1. Core Identity — 品牌人格

定義 AI 的「性格」，讓每一次互動都帶有品牌 DNA。

| 設定項 | 說明 | 範例 |
|--------|------|------|
| 角色定位 | AI 扮演什麼樣的角色 | 「科技宅的鄰家朋友」 |
| 語氣調性 | 說話的方式和感覺 | 「平易近人但有深度，偶爾冷幽默」 |
| 擅長主題 | 核心討論領域 | AI 工具、效率提升、數位生活 |
| 禁忌事項 | 絕對不能觸碰的內容 | 過度專業術語、居高臨下 |
| 招牌風格 | 讓人一眼認出品牌的特色 | 常說「試試看就知道了」 |
| 品牌聲音 | 詳細的聲音描述 | 像和科技圈的老朋友聊天 |
| 風格示例 | 典型的表達方式 | 具體的文案範例 |

### 2. Content Rules — 內容生成規則

精確控制 AI 產出的每一篇內容。

- **語言設定** — 繁體中文、簡體中文、英文，可附加語言備註
- **字數範圍** — 自定義最短/最長字數（如 150-300 字）
- **Emoji 策略** — 不使用 / 極少 / 適量 / 大量，可指定風格偏好
- **Hashtag 規則** — 不使用 / 少量 / 依平台而定
- **內容結構** — 敘事分享 / 列表整理 / 問答互動 / 混合
- **時段語氣調整** — 早晨溫暖、午間活力、晚間沉穩，5 個時段獨立設定

### 3. Customer Service — 客服規則

讓 AI 客服成為品牌最可靠的第一線。

- **問候模板** — 統一的開場白，建立品牌印象
- **回覆長度** — 控制客服回覆的字數範圍
- **轉人工觸發** — 定義哪些情境必須轉交真人處理
- **不可承諾事項** — AI 絕對不能答應的內容（退款、折扣等）
- **轉介資訊** — 需要人工時，提供正確的聯繫方式
- **FAQ 優先主題** — 最常被問到的問題，優先匹配回覆

### 4. Knowledge Base — 品牌知識庫

讓 AI 真正「認識」你的品牌。

- **基本資訊** — 品牌故事、核心產品、營業資訊
- **競品分析** — 了解市場定位，回覆時不踩雷
- **行業背景** — 產業脈絡，讓回覆更專業
- **活動日曆** — 季節性活動、促銷時間，自動配合宣傳

## Learning & Memory — 持續進化

Brand AI Agent 不是設定一次就結束，它會不斷學習和進化。

### 自動學習循環

```
發文 → 收集互動數據 → 分析高分內容 → 調整策略 → 更好的發文
  ↑                                                    |
  └────────────────────────────────────────────────────┘
```

- **高互動分析** — 自動追蹤哪些內容引發最多互動，學習成功模式
- **客戶記憶** — 記住每位客戶的偏好、歷史對話，提供個人化回覆
- **回覆優化** — 分析客服回覆效果，持續改善回應策略
- **主題追蹤** — 避免重複主題，保持內容新鮮度
