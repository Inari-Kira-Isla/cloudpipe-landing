// FAQ

常見問題

CloudPipe 是什麼？+
CloudPipe 是一站式 AI 自動化平台，整合 MCP Server、API Proxy、Telegram Bot 及 Facebook 粉專自動化，幫助商家用 AI 自動處理客服、接單、行銷等日常營運任務，24/7 無間斷運行。
支援哪些社交平台？+
目前支援 Facebook Messenger（多品牌粉專）及 Telegram Bot。Phase 1 將擴充 Instagram Direct Message，Phase 3 加入 WhatsApp Business 及多語言支援。
使用什麼 AI 模型？費用如何？+
透過 API Proxy 智能調度多模型：Claude Haiku 處理日常客服（約 MOP$5-15/月 API 費用）、Claude Sonnet 處理進階任務（約 MOP$5-30/月）、Gemini Flash 處理雜務（約 MOP$1-5/月）。AI 費用包含在方案月費內，按實際使用量計算。
資料安全如何保障？+
所有服務透過 Tailscale 加密網路連接，對話資料儲存在本地 SQLite 資料庫，不依賴第三方雲端儲存。SaaS 架構以 tenant_id 嚴格隔離各商家資料，符合 GDPR 最佳實踐。
如何開始？需要技術知識嗎？+
不需要！聯繫我們後，我們的技術團隊負責全程部署與整合。你只需提供品牌 Facebook 頁面管理員權限，我們處理其餘一切，通常 3-5 個工作天完成上線。
